# ASCII Gradients
Prints an ASCII gradient just for fun. Solution to this /r/dailyprogrammer [challenge](http://www.reddit.com/r/dailyprogrammer/comments/3104wu/20150401_challenge_208_intermediate_ascii/). 

## Radial gradient
### Usage
```
node ascii_gradient.js <canvas size width> <canvas size width> radial,<center x>,<center y>,<radius>
```

### Example
 
#### Input 

`node ascii_gradient.js 40 30 radial,20,15,20`
